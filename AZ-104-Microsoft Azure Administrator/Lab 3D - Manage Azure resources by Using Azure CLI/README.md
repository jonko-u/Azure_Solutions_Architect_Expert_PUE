
## Descripción del Laboratorio
Ahora que ha explorado las capacidades básicas de administración de Azure asociadas con el aprovisionamiento de recursos y su organización basada en grupos de recursos utilizando el portal de Azure, las plantillas de Azure Resource Manager y Azure PowerShell, necesita llevar a cabo la tarea equivalente utilizando Azure CLI. Para evitar instalar Azure CLI, aprovecharás el entorno Bash disponible en Azure Cloud Shell.

Objetivos

En este laboratorio, usted:

- Tarea 1: Iniciar una sesión Bash en Azure Cloud Shell.
- Tarea 2: Crear un disco gestionado Azure utilizando Azure CLI
- Tarea 3: Configurar el disco gestionado utilizando Azure CLI
