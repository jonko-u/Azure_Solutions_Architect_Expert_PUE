
## Descripción del Laboratorio
En este laboratorio, usted aprenderá acerca de las características básicas de almacenamiento, incluyendo cuentas de almacenamiento, almacenamiento blob, archivos Azure y File Sync, seguridad de almacenamiento y herramientas de almacenamiento.

Después de completar, usted será capaz de:

- Crear cuentas de almacenamiento Azure.
- Configurar contenedores blob.
- Proteger el almacenamiento Azure.
- Configurar los archivos compartidos de Azure y la sincronización de archivos.
- Administrar el almacenamiento con herramientas como Storage Explorer.
