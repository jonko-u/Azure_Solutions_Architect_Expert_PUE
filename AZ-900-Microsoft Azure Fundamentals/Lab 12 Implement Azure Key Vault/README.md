

## Descripción del Laboratorio
En este laboratorio, crearemos una bóveda de claves de Azure(Azure Key vault) y  luego crearemos una constraseña secreta dentro de esa bóveda de claves, proporcionando una contraseña almacenada de forma segura y administrada centralmente para su uso con aplicaciones.
